# Stock Advisor

LLM-assisted tool for fetching and visualizing stock data.

## Quick-start for contributors

1. **Create a virtual environment**

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. **Install dependencies**

```bash
pip install -e .[dev]
```

3. **Run tests**

```bash
pytest
```

4. **Run example**

```bash
python -m stock_advisor --help
```
